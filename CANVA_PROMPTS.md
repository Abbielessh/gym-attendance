# Canva Asset Prompts and Save Locations

Generate these assets in Canva and save them inside this folder:

```txt
public/assets/img/
```

Keep the exact file names below. The app already references `logo-gym.webp` and `bg-gym.jpg`.

## Required assets

### 1. App logo

Save as:

```txt
public/assets/img/logo-gym.webp
```

Canva prompt:

```txt
Professional modern logo for Kannai Fitness Studio, bold letter K combined with a dumbbell and attendance check mark, clean vector style, premium fitness studio brand, deep navy and electric blue gradient, minimal, transparent background, square 1024 by 1024, no emoji.
```

Export settings:

```txt
webp, transparent background, 1024 by 1024.
```

### 2. Full website background image

Save as:

```txt
public/assets/img/bg-gym.jpg
```

Canva prompt:

```txt
Premium modern gym interior background for Kannai Fitness Studio web dashboard, clean weight training area, soft blue and white lighting, professional commercial gym atmosphere, subtle blur, dark navy shadows, enough empty space for web cards, realistic but polished, no people facing camera, no text, no logo, no emoji, wide 16 by 9.
```

Export settings:

```txt
JPG, 1920 by 1080 or higher.
```

## Important image notes

Use simple names exactly as written. Do not add spaces in file names. Do not save as `.jpeg` if the app expects `.jpg`. For the background, the exact file must be:

```txt
bg-gym.jpg
```
